Title : EvergreenStudio

📌 Overview:
      EvergreenStudio is a modern and fully responsive front-end website built using Tailwind CSS. It is designed for showcasing natural flowers, artificial flowers, and indoor plants with a clean, elegant, and nature-inspired user interface. The project focuses on visual appeal, smooth responsiveness, and a seamless browsing experience across all devices.

✨ Features:
    Fully responsive design for all devices.
    Modern UI built with Tailwind CSS.
    Nature-inspired theme (flowers, plants, décor).
    Clean and minimal user interface.
    Fast loading and lightweight performance.
    Well-structured sections including products, reviews, and newsletter.

🛠️ Tech Stack:
    HTML5
    Tailwind CSS

🎯 Project Objective:
          This project was created to strengthen frontend development skills by building a real-world, responsive website using Tailwind CSS. It demonstrates practical UI design, layout structuring, and responsive web development techniques.
